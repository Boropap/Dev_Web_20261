# Website built for the WebDev class of 2026/1

---

> The following website is inspired by **Windows 95**. Its objective is to send a message that the internet of today is uninspired.
This repository is built in a simple structure where:

- _**`/fonts`**(Folder)_
    Has all the fonts used on the website
- _**`/icons`** (Folder)_
    Has all the icons used on the website
- _**`/images`** (Folder)_
    Has all the images used on the website
- _**`index.html`** (File)_
    Is the html file where the main website is located
- _**`style.css`** (File)_
    Is the styling using a css file

**The website can be seen by downloading and running the html. Or you can see online at** [World Wide Web](https://boropap.github.io/Dev_Web_20261/).
